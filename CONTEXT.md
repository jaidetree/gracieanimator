# Context & Glossary

The domain language for Gracie — the portfolio site for an animator/illustrator.
This file is a glossary only: no implementation details, no decisions (those live
in `docs/adr/`).

## Terms

### Project
An abstract grouping for a piece of portfolio work. Never instantiated directly;
always one of the concrete types below. Shares: **title**, **slug**, **order**,
**featured**, **published**.

### Storyboard
A project with a **category**, an optional rich **body**, a **thumbnail**, and
three ordered media sub-collections: **videos** (Vimeo/YouTube), **decks**
(Speakerdeck), and **PDFs**. Storyboards are the only content gated behind the
**storyboard password**.

### Video / Deck
An embed belonging to a Storyboard, authored as a provider URL (Vimeo/YouTube
for videos, Speakerdeck for decks). The provider's **oembed** is fetched when the
storyboard is saved, yielding the embed HTML, its dimensions, and (for videos) a
**poster** image.

### Poster
The provider-supplied still image for a video, fetched via oembed. Used as the
storyboard's thumbnail when none is uploaded.

### Comic
A project consisting of an ordered sequence of **comic pages** (images). Its
thumbnail is the first page.

### Illustration
A project that is a single image (also used as its own thumbnail).

### Sketchbook Sample
A project that is a single image (also used as its own thumbnail). Structurally
identical to an Illustration but a distinct type for display grouping.

### Page
A free-form standalone document (e.g. *about*, *resume*) with a **title**,
**slug**, and a rich **body**. Not a Project; not gated.

### Category
A grouping label for Storyboards only (e.g. shown at `/storyboards/category/<slug>/`).

### Body
The rich-text content of a Page, or the optional "More" section of a Storyboard.
Authored as HTML via a WYSIWYG admin widget; may contain inline images and
video embeds.

### Featured
A boolean marking a project for prominent display. The homepage shows one
featured project per type.

### Published
A boolean gate; only published projects/pages appear on the public site.

### Storyboard password
A single shared password that unlocks all storyboard pages for the duration of a
browser session.

### Thumbnail
The small representative image for a project shown in grids. Optional and
auto-derived when blank: a Comic uses its first page, an Illustration/Sketchbook
Sample uses its own image, a Storyboard uses its first video's **poster** (a
storyboard requires at least one video). A manually uploaded thumbnail always
wins; if none of the above is available (e.g. oembed hasn't produced a poster
yet), a CSS color block is shown. Smaller renditions are generated for grid
display.

### Slug
A URL-safe identifier for a project or page, auto-generated by slugifying the
title and unique within a type (e.g. `chicago-party-aunt`).
